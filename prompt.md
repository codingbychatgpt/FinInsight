# 任务目标
在【完全不改变现有业务逻辑、状态管理、接口调用和功能】的前提下，对 FinInsight-Frontend 进行全面的视觉美化与动效重构。采用国际前沿的“高级感极简金融风”，引入高帧率微动效与沉浸式排版，打造拿奖级别的 UI/UX。

# 核心设计规范 (Premium Financial Aesthetics)
- **视觉风格**：Bento Grid (便当盒网格布局) + Glassmorphism (高级毛玻璃质感)。
- **色彩升级**：引入微光（Bioluminescence）效果。主色调金融蓝（#1A56DB）增加霓虹流光遮罩；卡片背景采用 `rgba(255, 255, 255, 0.75)` 结合 `backdrop-filter: blur(12px)`；底色使用极具呼吸感的冷灰（#F4F6F9）。
- **动效哲学**：统一使用高级贝塞尔曲线 `cubic-bezier(0.25, 1, 0.5, 1)`（缓入疾出，极具物理惯性）。拒绝生硬闪现。

# 核心开发需求 (无损重构 tsx 结构与 scss 样式)

## 1. 首页“同步按钮”与全局状态动效
- **呼吸发光**：同步按钮未触发时，边缘带有极弱的“金融蓝”光晕呼吸效果（柔和阴影动画）。
- **激活动效**：点击同步（Loading 状态）时，按钮平滑缩放（`scale(0.95)`），内部 Loading 图标转变为多重无缝旋转的太极圈或极光流转线，按钮背景呈现流光渐变（Shimmer Effect）动画。
- **完成反馈**：同步结束、数据弹出的刹那，按钮触发一次微弱的向外涟漪扩散（Ripple Effect）波纹动画。

## 2. 资讯列表：瀑布流交错飞入 (Staggered Animation)
- **骨架屏升华**：数据加载前，骨架屏（Skeleton）不是简单的闪烁，而是带有从左到右微亮扫过的“拉丝银”流光动画。
- **进场动画**：新闻列表卡片加载时，**绝对不能**一起蹦出来。必须使用 CSS 变量赋予交错延迟（Staggered Delay），前 5 条卡片以 60ms 的间隔，从下方 20px 处伴随透明度 0 逐渐平滑滑入（Fade-in & Slide-up），速度曲线要像丝绸般顺滑。
- **悬停微交互 (H5端自动激活)**：手指或鼠标悬停在卡片上时，卡片平滑上浮 4px，阴影由硬变柔（扩散范围增大），卡片内的文本标题颜色微妙渐变，右下角隐藏的微型箭头或图标平滑向右横移 3px 浮现。

## 3. 影响力分数 (Impact Score) 视觉视觉锤
- 1-10分的分数不再是普通的数字，将其重构为一个带有能量感的设计：
  - 高分（8-10）包裹在微型的红色极光渐变圈内。
  - 页面加载时，分数的背景色块或圆形外圈有一个从 0 放大到 1 并在结尾处带有轻微弹性抖动（Elastic Pop）的动效。

## 4. 详情页：“双视角面板”沉浸式排版与长文本折叠
- **毛玻璃悬浮条**：详情页顶部导航或文章标题区升级为半透明悬浮毛玻璃，随页面下滑紧贴顶部，背景完美模糊。
- **双视角卡片对撞**：
  - 大众视角卡片（Public View）：加入极其细腻的浅蓝到浅白渐变纹理，段落首字平滑放大，带有金融杂志排版的高级感。
  - 银行人视角卡片 (Banker View)：采用深邃高奢的暗曜黑/深钛灰撞色，边缘带有一条微弱的 1 像素内发光（Inner Shadow），凸显“密室决策”的专业沉浸感。
- **原文平滑折叠**：长文本 `raw_content` 的折叠与展开，抛弃生硬的隐藏显示，使用 `max-height` 或 `grid-template-rows` 配合 `transition: all 0.5s` 达成丝滑的卷轴式收放动效，并在折叠处覆盖一层高质感的半透明线性渐变蒙版。
