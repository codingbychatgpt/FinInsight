# 任务目标
在【完全不改变现有任何业务逻辑、State 状态、API 接口调用和数据流】的前提下，将整个 FinInsight-Frontend 彻底重构为纯 Web/PC 端（Desktop-First）拿奖级别的全栈视觉。摆脱所有小程序束缚，利用大屏优势进行多栏响应式与便当盒（Bento Grid）布局重构。

# 👑 视觉与动效规范 (Premium Web Aesthetics)
- **核心风格**：Stripe / Vercel 级别的暗黑高奢金融风。底色采用深邃夜空黑（#060814），卡片采用曜石半透明黑（rgba(15,23,42,0.65)）配合 `backdrop-filter: blur(20px)`。
- **环境光晕**：引入高级网页标配的动态朦胧流光（Ambient Glow），在页面背景使用绝对定位的大面积低饱和度径向渐变，营造科技下沉感。
- **物理动效**：全场统一使用苹果御用缓动曲线 `cubic-bezier(0.16, 1, 0.3, 1)`。强制开启 GPU 硬件加速（多用 transform3d 和 opacity），绝不允许引起网页重排（Reflow）导致的卡顿。

# 🛠️ 页面级重构需求

## 1. 全局架构与首页 (`src/pages/index`) 重构
- **版面大改版**：废弃单列堆叠。在 PC 宽屏上，采用多列 Bento Grid 瀑布流布局。
- **顶部控制台**：
  - “同步最新政策”按钮升级：未激活时自带金融蓝（#3B82F6）弱光晕呼吸阴影；点击时平滑微缩并触发骨架流光渐变（Shimmer Effect）；同步完成的瞬间向外扩散一层微弱的涟漪（Ripple Wave）。
- **新闻流卡片飞入 (Staggered Load)**：
  - 骨架屏：数据加载前，展示带有从左到右扫过的“拉丝银”渐变动画。
  - 进场：卡片加载时引入交错延迟（利用 CSS var 分配 delay），以 60ms 间隔平滑滑入。
  - 悬停微交互：鼠标 hover 时卡片平滑上浮 4px，阴影由硬变柔，卡片边缘 1 像素内发光亮起，右下角隐现微型箭头并右移 3px。

## 2. 详情页 (`src/pages/detail`) 硬核重构
- **黄金比例双栏布局 (Desktop Split Panel)**：
  - **左侧栏 (35% 宽度, Sticky 常驻)**：展示媒体来源徽标、发布日期与大标题。下方悬浮核心视觉锤——“数字巨幕仪表盘”，将 `impact_score`（影响力分数）放大至巨幕尺寸。8-10分包裹在红外线极光渐变圈内，5-7分包裹在晶莹橙圈内。
  - **右侧栏 (65% 宽度, 滚动流)**：作为便当盒容器。
- **双视角对撞卡片**：
  - 大众视角卡：采用浅蓝到浅白的深邃渐变纹理，段落首字下沉放大，具备顶尖金融杂志的排版质感。
  - 银行人视角卡：采用极致曜石黑，边缘带有一条微弱的 1 像素同业绿（#10B981）内发光，凸显机构级专业感。
- **官方原文高性能折叠**：
  - 针对千字长文，**严禁**使用引发重排卡顿的 `max-height` 动画。
  - 必须采用现代 CSS Grid 动画引擎（`grid-template-rows: 0fr -> 1fr` 结合 `translate3d`）实现无损丝滑卷轴式收放，折叠处覆盖半透明线性渐变蒙版。

## 3. 响应式与代码纯净度
- 仅允许通过调整 TSX 中的 `className` 和增加必要的视觉装饰性 `View` 容器（如背景流光、卡片饰条）来进行视觉升级，不得改动任何数据逻辑。
